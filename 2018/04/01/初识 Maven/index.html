发表于 2018-04-01 | 分类于 编程实践 |

相信从事 Java 开发的小伙伴们对 Maven 这个词都不陌生。目前来说，只要存在外部依赖的 Java 项目基本上都是 Maven 项目，而每个 Maven 项目都有一个明显的特征——pom.xml 文件。

今天，我们就一起来学习一下什么是 Maven 项目。
